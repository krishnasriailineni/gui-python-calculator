# 🧮 GUI Calculator (Python)

A simple and interactive "GUI-based Calculator" developed using Python. This application provides a clean and user-friendly interface to perform mathematical calculations efficiently.

---

## 🚀 Features
- ➕ Addition  
- ➖ Subtraction  
- ✖ Multiplication  
- ➗ Division  
- 📊 Modulo  
- 🔢 Power  
- ⬇ Floor Division  
- 🖥️ Easy-to-use graphical interface  

---

## 🛠️ Technologies Used
- Python  
- Tkinter (for GUI development)  

---

## 📌 How to Run

1. Install Python on your system  
2. Download or clone this repository  
3. Open terminal in the project folder  
4. Run the program using:

```bash
python calculator.py
